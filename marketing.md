# Marketing Plan — Claude Code Inspect

## Core message

> You don't need design tools anymore.

Claude can now see your UI, understand your components, and help you fix them — just by clicking an element. No copy-pasting selectors. No describing layouts. No switching context. Just click, prompt, done.

> Stop wasting tokens describing what Claude could just see.

Most developers burn hundreds of tokens per session just setting up context — pasting selectors, explaining layouts, attaching screenshots as base64 blobs. Claude Code Inspect eliminates all of that. Claude gets the selector, component name, source file, and screenshot automatically. Screenshots are delivered natively to Claude's vision encoder and only when Claude actually needs them — zero tokens when it can answer from code alone.

---

## Target audience

- Frontend developers who use Claude Code daily
- React / Vue / Angular developers debugging UI issues
- Developers who are frustrated switching between browser devtools, Figma, and their terminal
- Early adopters of AI-assisted development workflows

## Store listing description

```
Claude Code Inspect connects your browser to Claude Code, giving Claude instant, full context about any element you click — no copy-pasting required.

HOW IT WORKS
Click the Inspect button, then click any element on the page. Claude receives the element's CSS selector, dimensions, React component name, source file, and a screenshot — automatically. Prompt Claude and it answers directly in your terminal.

WHAT YOU GET
• Zero copy-pasting — Claude sees exactly what you're looking at
• React-aware — knows the component name and source file
• Token-efficient — screenshots are only sent to Claude's vision encoder when needed, never bloating your context
• Works on any website — not just apps you own

REQUIREMENTS
• Claude Code (claude.ai/code) installed and running
• Bun runtime for the companion plugin

WHY IT'S DIFFERENT
Most AI coding tools make you describe your UI in words. Claude Code Inspect makes that unnecessary. One click gives Claude everything — and Claude jumps straight to the solution.
```
